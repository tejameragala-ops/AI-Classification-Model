# AI Data Classification Using Machine Learning

## Project Overview
This project is a basic Artificial Intelligence classification model built using Python and Machine Learning.

The model uses the Iris Flower Dataset to classify flower species based on flower measurements.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NumPy

## Features
- Dataset Loading
- Data Preprocessing
- Train-Test Split
- Model Training
- Prediction System
- Accuracy Evaluation

## Machine Learning Algorithm
Decision Tree Classifier

## Output
- Predicts flower category
- Gives model accuracy
